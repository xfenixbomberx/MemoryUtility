# Memory Utility

Memory Utility is a lightweight Windows memory cleaning tool designed to quickly reclaim unused memory with minimal overhead.

## Features
- One-click memory cleaning
- Optional advanced cleaning (standby list, system working set)
- System tray support
- Automatic cleaning modes
- Low resource usage
- Portable, single EXE installer

## Installation
1. Download **MemoryUtilitySetup.exe** from the Releases page
2. Run the installer
3. If Windows SmartScreen appears:
   - Click **More info**
   - Click **Run anyway**

The application is unsigned to keep distribution free and simple.

## Updates
Memory Utility can check for updates automatically and will notify you when a new version is available.

## Security & Transparency
- No malware
- No bundlers
- No background downloads
- No telemetry

SHA-256 checksums are provided with every release so downloads can be verified.
